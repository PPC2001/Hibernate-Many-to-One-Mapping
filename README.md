# Hibernate-Many-to-One-Mapping
In hibernate , many to one mapping, various attributes can be referred to one attribute only.
In this example, many reviews belongs to one product


Tools and Technologies used
--------------
1. Eclipse IDE
2. Maven 4.0
3. MySQL 8.0
4. Hibernate
5. Java
